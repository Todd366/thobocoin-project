# THoBoCoin Deployment Checklist

Complete checklist for launching THoBoCoin.

## Pre-Deployment

### Development
- [ ] Node.js installed
- [ ] Dependencies installed
- [ ] Contracts compiled
- [ ] All tests passing
- [ ] Environment variables set

### Security
- [ ] Contracts reviewed
- [ ] Access controls verified
- [ ] Consider professional audit

### Wallet
- [ ] Deployer wallet created
- [ ] Private key secured
- [ ] Sufficient BNB for gas
- [ ] Wallet backed up

## Testnet Deployment

### Deploy
- [ ] Get testnet BNB
- [ ] Run: `npm run deploy:testnet`
- [ ] Save contract addresses
- [ ] Verify on BscScan

### Test
- [ ] Update frontend addresses
- [ ] Test wallet connection
- [ ] Test transfers
- [ ] Test staking
- [ ] Test governance
- [ ] Create test rooms

### Community
- [ ] Share with testers
- [ ] Collect feedback
- [ ] Fix issues
- [ ] Document problems

## Mainnet Preparation

### Final Checks
- [ ] Testnet successful
- [ ] Audit completed (optional)
- [ ] Team roles assigned
- [ ] Emergency plan ready

### Marketing
- [ ] Website live
- [ ] Social media created
- [ ] Whitepaper published
- [ ] Launch announcement ready
- [ ] 100+ community members

### Liquidity
- [ ] Initial amount decided
- [ ] PancakeSwap plan ready
- [ ] Marketing budget allocated

## Mainnet Deployment

### Deployment Day
- [ ] Final configuration check
- [ ] Sufficient BNB (~0.1)
- [ ] Run: `npm run deploy:mainnet`
- [ ] **SAVE ALL ADDRESSES**
- [ ] Verify contracts
- [ ] Test basic functions

### Distribution
- [ ] Allocate tokens per tokenomics
- [ ] Transfer to wallets
- [ ] Set team vesting
- [ ] Announce transparency

## Post-Deployment

### Liquidity
- [ ] Add to PancakeSwap
- [ ] Test trading
- [ ] Announce pair address

### Integration
- [ ] Create BSTM rooms
- [ ] Assign managers
- [ ] Test contributions

### Frontend
- [ ] Update with mainnet addresses
- [ ] Deploy to hosting
- [ ] Test all functions

### Marketing
- [ ] Launch announcement
- [ ] Twitter/Telegram posts
- [ ] Update documentation
- [ ] CoinMarketCap application
- [ ] CoinGecko application

## Week 1

### Community
- [ ] Daily engagement
- [ ] Answer questions
- [ ] Share updates

### Technical
- [ ] Monitor transactions
- [ ] Check for exploits
- [ ] Verify functions
- [ ] Monitor liquidity

### Marketing
- [ ] Daily posts
- [ ] Influencer outreach
- [ ] AMA sessions
- [ ] Partnership announcements

## Month 1 Goals

- [ ] 1,000+ holders
- [ ] $50K+ liquidity
- [ ] 5+ active rooms
- [ ] 10+ proposals
- [ ] First DEX listing

## Emergency Procedures

### Contract Issue
1. [ ] Pause if severe
2. [ ] Notify community
3. [ ] Assess with team
4. [ ] Deploy fix
5. [ ] Post-mortem

### Price Volatility
1. [ ] Monitor manipulation
2. [ ] Communicate calmly
3. [ ] Check liquidity
4. [ ] Review for exploits
